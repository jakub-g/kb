getBoundingClientRect
It's the box as seen in Firebug when inspecting the element.
Numbers returned depend on the scroll of the element and all parent elements.

If right/bottom is negative, then object is out of viewport.
If left/top is negative, then object is partially out of viewport.
clientLeft
In left-to-right environments, it's equal to border-left-width (otherwise it also includes scrollbar width)
Note though that for inline elements, clientLeft is always zero.
offsetLeft
Left position with respect to offsetParent (nearest ancestor with position:absolute or relative).
